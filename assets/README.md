# Assets

Diese PNG-Dateien sind schlichte, einfarbige Platzhalter (Foreverly-Gold &
Cremeweiß), damit die App sofort startet. Bitte vor Veröffentlichung durch
finale Markenassets ersetzen:

- `icon.png` (1024×1024) – App-Icon
- `adaptive-icon.png` (1024×1024) – Android Adaptive Icon (Vordergrund)
- `splash.png` – Splash-/Startbildschirm
- `notification-icon.png` (96×96) – Benachrichtigungs-Icon (Android)
